# LGPD e Privacidade

## Dados tratados

- Dados cadastrais.
- Dados de jornada.
- Geolocalização.
- Evidências de batida.
- Selfie, quando ativada.
- Documentos.
- Assinaturas.
- Logs de acesso.

## Princípios

- Transparência.
- Minimização.
- Finalidade.
- Segurança.
- Retenção controlada.
- Auditoria de acesso.
- Telemetria somente durante jornada ativa, quando configurada.
