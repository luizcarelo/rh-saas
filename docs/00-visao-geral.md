# RH SaaS Brasil — Visão Geral

Esta documentação descreve a evolução do RH SaaS para uma plataforma completa de gestão de recursos humanos, ponto eletrônico, documentos, holerites, banco de horas, justificativas, telemetria e app mobile offline-first.

## Objetivos

- Manter arquitetura multitenant.
- Criar ambiente Super Admin para gestão de clientes SaaS.
- Evoluir frontend-web sem quebrar funções existentes.
- Evoluir backend em fases.
- Evoluir mobile app com funcionamento offline, GPS preciso e sincronização posterior.
- Manter rastreabilidade e auditoria.
- Preparar base futura para compatibilidade AFD/REP-P.
- Manter documentação atualizada a cada fase.

## Ordem de implementação

1. Banco de dados.
2. Backend.
3. Frontend-web.
4. Mobile app.
5. Testes e auditoria.
