# Super Admin

O ambiente Super Admin será responsável por:

- Criar clientes.
- Ativar/desativar módulos.
- Gerenciar planos.
- Acompanhar status de clientes.
- Ver indicadores operacionais.
- Acessar trilhas de auditoria administrativas.
