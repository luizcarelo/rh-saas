# Modelo de Dados

Esta fase cria tabelas fundacionais adicionais, sem remover ou quebrar tabelas existentes.

## Grupos de tabelas

- SaaS/Super Admin.
- Estrutura RH.
- Ponto mobile.
- Documentos.
- Holerites.
- Telemetria.
- Auditoria.
- Dados de teste.

## Regra de segurança

Nenhuma batida original deve ser alterada diretamente. Ajustes devem gerar novos registros de ajuste/justificativa.
