# Ponto e AFD

## Premissas

- Preservar marcações originais.
- Criar NSR interno.
- Registrar origem da marcação.
- Guardar hash de integridade.
- Ajustes não substituem marcações brutas.

## Tipos principais

- Entrada.
- Saída.
- Início de Intervalo.
- Fim de Intervalo.

## Tipos opcionais

- Hora Extra.
- Visita Externa.
- Sobreaviso.
- Plantão.
- Pausa Técnica.
- Saída Particular.
- Viagem.
