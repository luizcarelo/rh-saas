# Mobile App

## Objetivo

Criar app offline-first para marcação de ponto, consulta de histórico, espelho, banco de horas, justificativas, documentos, assinatura e holerites.

## Regras principais

- Login usando `access_token`.
- API via `EXPO_PUBLIC_API_URL`.
- GPS preciso obrigatório quando configurado.
- Exibição transparente do endereço antes da batida.
- Batida offline com sincronização posterior.
- Tela de pendências de sincronização.
- Selfie opcional por política administrativa.
- Telemetria apenas durante jornada ativa, quando configurada.
