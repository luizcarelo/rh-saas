# Módulo Funcionário — Escopo oficial

## Escopo

O módulo Funcionário será o núcleo operacional do RH SaaS.

Nesta etapa, o trabalho será restrito a:

- backend;
- app admin web;
- banco de dados.

O app mobile não será implementado agora, mas o backend e o banco deverão ser preparados para consumo futuro pelo mobile.

## Direção de produto

O cadastro de funcionário deverá ser progressivo e modular.

Não deverá ser criado um formulário único gigante.

## Áreas futuras do módulo

- cadastro mínimo;
- dados pessoais;
- endereço;
- contrato;
- jornada e ponto;
- documentos;
- dependentes;
- benefícios;
- financeiro e bancário;
- saúde e segurança;
- vínculo com usuário;
- auditoria;
- pendências e notificações.
